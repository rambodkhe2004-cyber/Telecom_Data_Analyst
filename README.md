# Telecom_Data_Analyst

# 📊 Telecom Data Analysis Dashboard (Power BI)

## 📌 Overview

This project presents a comprehensive **Telecom Data Analysis Dashboard** built using Power BI. It provides insights into customer behavior, revenue trends, churn analysis, and service usage patterns to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze telecom customer data to identify key trends
* Monitor KPIs such as revenue, churn rate, and customer growth
* Understand customer segmentation and usage behavior
* Provide actionable insights to improve customer retention

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV** – Data source
* **DAX** – Data calculations and measures
* **Power Query** – Data cleaning and transformation

---

## 📂 Dataset Description

The dataset includes telecom-related information such as:

* Customer demographics (age, gender, location)
* Subscription details (plan type, tenure)
* Usage metrics (call duration, data usage)
* Billing & revenue data
* Churn status



## 📊 Dashboard Features

* **KPI Cards**: Total Customers, Revenue, Churn Rate
* **Customer Insights**: Segmentation by demographics and plan types
* **Revenue Analysis**: Monthly trends and plan-wise revenue
* **Churn Analysis**: Identifying factors affecting customer churn
* **Usage Patterns**: Call and data usage behavior



## 🔍 Key Insights

* Identified high churn segments based on contract type and tenure
* Highlighted top revenue-generating plans
* Observed customer usage patterns for targeted marketing
* Provided insights for improving retention strategies
